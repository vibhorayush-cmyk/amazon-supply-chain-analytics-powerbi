# Amazon Supply Chain Analytics - Power BI

This repository contains SQL scripts and supporting files for the Amazon Supply Chain Analytics Power BI solution.

## Repository structure

- `PowerBI/`
  - `Amazon Dashboard.pbix`
- `SQL/`
  - `Fact table.sql`
  - `Dim Customer.sql`
  - `Dim Date.sql`
  - `Dim Shipping.sql`
  - `Dim_product.sql`
  - `New_Query_1781556304168.sql`
- `Images/`
  - `ExecutiveDashboard.png`
  - `ProductAnalytics.png`
  - `ProductDrillthrough.png`

## Notes

- Add `Amazon Dashboard.pbix` to the `PowerBI/` folder.
- Add the dashboard screenshots to the `Images/` folder.
- The SQL scripts are stored in the `SQL/` folder.

## About

This project is an analytics solution for Amazon supply chain data using Power BI and SQL.
